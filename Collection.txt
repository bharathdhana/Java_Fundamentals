ArrayList:
add elements - add()
remove elements - remove()

LinkedList:
add elements - offer()
remove elements - poll()

Note:
LinkedList can also be used as a stack (LIFO) or a queue (FIFO).

stackOperations:
add - push()
remove - pop()

HashMap:
keySet() -> Set<String>
values() -> Collection<String>
entrySet() -> Set<Map.Entry<String, String>>

TreeMap:
add -  put()
